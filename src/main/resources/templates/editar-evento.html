Atualizar Informações do Evento

Resumo das funcionalidades: Permite editar todos os dados de um evento, como título, descrição, data, local e link. Os campos do formulário são preenchidos automaticamente com os dados atuais do evento usando Thymeleaf. O botão "Salvar Alterações" envia o formulário para o backend atualizar o evento. Botões para voltar à lista de eventos e para sair/logout no topo da página. Relacionado Quais elementos do formulário podem facilitar a confirmação de dados antes de salvar
